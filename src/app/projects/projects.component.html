REMOTE CONTROL AND MONITORING IoT SYSTEM

I developed a web-based IoT control and monitoring system deployed on a Raspberry Pi, integrating various sensors and actuators such as a DHT11 temperature sensor, LED controller, and a gas sensor. The system is managed through an intuitive web interface, built using HTML, CSS, JavaScript, and Bootstrap, enabling users to:

  • Monitor real-time temperature data from the DHT11 sensor.
  • Remotely control the LED, turning it on or off.
  • Monitor gas levels and receive alerts if dangerous gas levels are detected.

The Raspberry Pi serves as the central hub, with Firebase handling real-time data synchronization between the sensors and the web interface. The backend, powered by Python and Flask, facilitates communication between the web interface and the hardware components. This dynamic system provides efficient, real-time monitoring and control, making it a powerful IoT solution for remote operations.

DESIGN AND DEVELOPMENT OF A WEB APPLICATION FOR CLUBS

Developed a full-stack web application using HTML, CSS, Python, Django, and Bootstrap for clubs management. Key features of the project include:

  • Backend development with Django, utilizing SQLite for database operations to manage club activities.
  • User authentication and role-based access control, ensuring secure and tailored access to different features based on user roles (administrators, members).
  • Creation of CRUD functionalities (Create, Read, Update, Delete) for handling club events, members, and resources.
  • Implementation of a responsive web design using Bootstrap, ensuring the interface adapts seamlessly across various devices for an optimal user experience.

This project demonstrates strong proficiency in web technologies, combining front-end design with back-end logic for a fully integrated web application.

STICKER SELLING PLATFORM - STICKY TICKY

Developed a dynamic web platform for the sale of customized stickers, featuring comprehensive management of users, products, and orders. Key highlights of the project include:

  • Utilized the MEAN stack(MongoDB, Express.js, Angular, Node.js) to deliver a full-stack architecture, ensuring smooth and efficient performance across the platform.
  • Implemented a responsive and interactive front-end using Angular, providing users with a seamless browsing and shopping experience.
  • Built a robust back-end with Node.js and Express.js, handling user authentication, product management, and order processing.
  • Managed real-time data and transactions with MongoDB, allowing efficient and scalable data handling for users, products, and orders.

This project showcases the ability to create a fully integrated e-commerce platform, emphasizing both user experience and scalable architecture.

Automated Medical Diagnosis System

Creation of a user-friendly HMI interface enabling healthcare professionals to automatically diagnose diseases from medical images, such as X-rays, MRIs, and scans, by leveraging convolutional neural networks (CNNs). This includes an interactive dashboard for seamless image uploading, AI model processing, and presenting diagnostic results with detailed annotations.
