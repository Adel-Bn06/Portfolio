REMOTE CONTROL AND MONITORING IoT SYSTEM

I developed a web-based IoT control and monitoring system deployed on a Raspberry Pi, integrating various sensors and actuators such as a DHT11 temperature sensor, LED controller, and a gas sensor. The system is managed through an intuitive web interface, built using HTML, CSS, JavaScript, and Bootstrap, enabling users to:

  • Monitor real-time temperature data from the DHT11 sensor.
  • Remotely control the LED, turning it on or off.
  • Monitor gas levels and receive alerts if dangerous gas levels are detected.

The Raspberry Pi serves as the central hub, with Firebase handling real-time data synchronization between the sensors and the web interface. The backend, powered by Python and Flask, facilitates communication between the web interface and the hardware components. This dynamic system provides efficient, real-time monitoring and control, making it a powerful IoT solution for remote operations.

MINI-PROJECT ON CIRCUIT DESIGN WITH ALTIUM

This project involved the development of a complex circuit design using Altium Designer A17, which consisted of 6 detailed schematics and incorporated a wide range of components. These included inductors, transistors, integrated circuits (ICs), and various passive components. The design process required careful planning and execution to ensure proper functionality and integration of the circuit elements. The project also focused on optimizing the layout for efficiency and performance, leveraging Altium's advanced features to handle the complexity of the circuit while ensuring seamless interconnections between the components. The diagram highlights a power control system with multiple stages, including signal amplification, switching mechanisms, and power regulation, demonstrating a high level of precision in component placement and schematic design.
